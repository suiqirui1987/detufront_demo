detufront_demo
=====================================

## 安装fis-pure

```bash
npm install -g fis-pure
```

安装成功后执行 ``pure -h`` 即可看到相关开发命令帮助

## 示例

首先，启动内置的调试服务器：

```bash
pure server start
```


```bash
cd detufront_demo
```

第三步，执行fis的编译命令：

```bash
pure release
```

第四步，刷新浏览器，查看我们的项目。

可以看到所有的模块化资源均已自动加载进来了
